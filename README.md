# covid19_webapp_bulgaria
Python web app built on Dash to track Covid-19 spread in Bulgaria.

Data is periodically updated with latest information from the Bulgarian Ministry of Health - http://www.mh.government.bg/bg/informaciya-za-grazhdani/potvrdeni-sluchai-na-koronavirus-na-teritoriyata-na-r-blgariya/
